# VSCode iCalendar Editor

iCalendar (.ics) ファイルを Visual Studio Code 上で GUI を用いて視覚的に表示・編集するための拡張機能です。

## 主な機能

- **カレンダー表示**: イベントをカレンダー形式で一覧表示。
- **イベント編集**: イベントの作成、更新、削除。
- **テキスト表示切替**: 右上のアイコンから通常のテキストエディタ（iCal 形式）と GUI エディタを切り替え可能。

## 使用方法

1. VSCode で `.ics` 拡張子のファイルを開きます。
2. デフォルトで GUI エディタが表示されます。
3. イベントをクリックして詳細の確認や編集が可能です。
4. カレンダー内の空きスペースをクリックして新しいイベントを追加できます。

## 開発方法

### セットアップ

```bash
npm install
```

### コンパイル

```bash
npm run compile
```

### デバッグ

F5 キーを押して「Extension Development Host」を起動し、テスト用の `.ics` ファイルを開いて動作を確認してください。
